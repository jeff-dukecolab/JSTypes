Data Types

What are they?

types of data!

A data type specifies exactly what `type` of data a variable has. The types are:

  • Number - 1,2,3
  • String - "I am a string"
  • Boolean - True or false
  • Undefined - unset or unknown data
  • NULL - a known variable that has no value
  • Array - ["up", "down", "left", "right"]
  • Object - {"key": "value", "another key": "another value"}

How do I use them?

Any way you want!
